Build notes for handoff

  • Background: the header art is a real shipping lane (origin to destination, gold pin = delivered), inline SVG. No photoshoot needed, fully on visual-system. One bold gesture per surface.
  • Lift zones are marked in the code with LIFT ZONE comments: swap initials for real teammate photos, and replace Maya R. with a real name in v2.
  • Tokens: blue #4C62EA, gold #facc0b, dark #1a1d23. Type: Plus Jakarta Sans (display), Inter (body), JetBrains Mono (utility).
  • Intercom mapping: header art and greeting go in the Messenger home config, the two cards map to a "Send us a message" action and a custom link/CTA. v2 needs teammate routing turned on so the named person is real.
  • Honesty check: keep "no bots" only while a human truly answers first. If routing ever changes, the copy changes with it.
